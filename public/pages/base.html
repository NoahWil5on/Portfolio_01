Base Zero

Base Zero is a Zombie First Person Shooter game built in Unity and C#. Base Zero features a dynamic quest system, customizable and upgradeable weapons, diverse levels, and a plethera of in-house art. The project is still in development and we plan on releasing the game on steam. I am responsible for all player related mechanics (shooting, moving, health and resource management) as well as designing the framework to allow for limitless weapon customization, weapon feel, and weapon types.
