Shader Gallery

Showcase of a DirectX11 game engine built in C++. Designed and implemented bloom shader, shadow shader, normal map shader. Engine was built from scratch and features model loading, collision detection system, as well as an entity/texture/material/model management system. While not particularly aesthetic, the project serves as a demonstration of how these graphical elements can be implemented at a low level, and how the render pipeline works as a whole.

Realtime shadow mapping implementation

Demonstration of texture, normal, and specular mapping all at once

Bloom demonstration, notice how the bright light bleeds outside on the right (very high bloom values to make more obvious)

Simple UI that the mouse can interact with

Particle system implemented using a particle shader
